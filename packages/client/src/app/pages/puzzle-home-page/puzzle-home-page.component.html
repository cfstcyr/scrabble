Mode puzzle

Un plateau de jeu et un chevalet est généré. Vous avez un tour afin de créer le meilleur placement possible. Pour chaque puzzle, il existe toujours au moins un placement utilisant toutes les tuiles. Votre objectif est de le trouver !

Le mode "Puzzle du jour" présente un seul jeu par jour. Faites le meilleur placement possible pour gagner une place dans le classement ! Un nouveau puzzle est disponible à tous les jours à 8h P.M.
